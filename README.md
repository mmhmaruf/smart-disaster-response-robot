# 🤖 Smart Disaster Response Robot

## 🚀 Overview

An AI-powered robot that detects disaster situations using TinyML and multiple sensors.

## 🔥 Features

* Sound-based help detection
* Vibration detection
* Human presence detection
* Autonomous decision system

## 🧠 AI Logic

* Help → Move
* Vibration → Stop + Alert
* Human → Stop
* No event → Explore

## 🧰 Hardware

ESP32, INMP441, MPU6050, L298N
