🤖 Smart Disaster Response Robot

AI-powered Robot using TinyML + ESP32

🚀 Overview

This robot detects disaster situations using sound, vibration, and AI. It can autonomously respond and assist in rescue operations.

🧠 Features

  • 🎤 Help sound detection
  • 📡 Vibration detection
  • 🤖 Autonomous movement
  • ⚠️ Alert system (buzzer)
  • 🧠 AI decision making

🧠 AI Logic

Help detected → Move forward
Vibration → Stop & Alert
Person → Stop
No input → Explore mode
